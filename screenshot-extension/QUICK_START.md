# 快速开始指南

## 第一步：生成插件图标

1. 在浏览器中打开 `images/generate-icons.html`
2. 图标会自动下载到你的下载文件夹
3. 将下载的 `icon-16.png`、`icon-48.png`、`icon-128.png` 放入 `images/` 文件夹

## 第二步：安装插件

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 启用右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `screenshot-extension` 文件夹
6. 完成！

## 第三步：使用插件

1. 访问任何网站
2. 点击浏览器右上角的插件图标（相机图标）
3. 选择截图类型：
   - **全页面截图** - 截图整个网页
   - **可见区域截图** - 截图当前视口
   - **选定区域截图** - 拖拽选择区域

## 截图保存位置

所有截图都会自动保存到你的 **下载文件夹**，文件名格式为：
```
screenshot-{type}-{timestamp}.png
```

例如：
- `screenshot-fullpage-2024-01-15T10-30-45.png`
- `screenshot-visible-2024-01-15T10-30-50.png`
- `screenshot-select-2024-01-15T10-30-55.png`

## 常见问题

**Q: 插件图标没有显示？**
A: 确保已将图标文件放入 `images/` 文件夹，然后在 `chrome://extensions/` 中刷新插件。

**Q: 某些网站截图失败？**
A: 某些网站有跨域限制。可以尝试使用"可见区域截图"功能。

**Q: 如何卸载插件？**
A: 在 `chrome://extensions/` 中找到插件，点击 **移除** 按钮。

## 开发和调试

### 查看控制台日志
1. 在 `chrome://extensions/` 中找到插件
2. 点击 **详情**
3. 点击 **检查视图** 下的 **service worker**

### 修改代码后重新加载
1. 在 `chrome://extensions/` 中找到插件
2. 点击刷新按钮
