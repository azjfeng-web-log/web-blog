# 页面截图 Chrome 插件

一个功能强大的 Chrome 浏览器截图插件，支持全页面、可见区域和选定区域截图。

## 功能特性

- 📄 **全页面截图** - 截图整个网页（包括滚动区域）
- 👁️ **可见区域截图** - 只截图当前可见的部分
- ✂️ **选定区域截图** - 自由选择要截图的区域
- 💾 **自动保存** - 截图自动保存到下载文件夹
- 🎨 **美观界面** - 现代化的 UI 设计

## 安装方法

### 方法一：开发者模式安装

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 启用右上角的 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `screenshot-extension` 文件夹
5. 完成！插件将出现在扩展程序列表中

### 方法二：打包安装

1. 在 `screenshot-extension` 文件夹右键选择 **压缩**
2. 将 `.zip` 文件重命名为 `.crx`
3. 在 Chrome 中打开 `chrome://extensions/`
4. 将 `.crx` 文件拖入浏览器窗口

## 使用方法

1. 点击浏览器右上角的插件图标
2. 选择截图类型：
   - **全页面截图** - 截图整个网页
   - **可见区域截图** - 截图当前视口
   - **选定区域截图** - 用鼠标拖拽选择区域
3. 截图自动保存到下载文件夹

## 技术栈

- **Manifest V3** - 最新的 Chrome 扩展程序标准
- **html2canvas** - 用于全页面和选定区域截图
- **Chrome APIs** - 用于可见区域截图和文件下载

## 功能详解

### 全页面截图
使用 `html2canvas` 库将整个 DOM 树转换为 Canvas，支持截图超出视口的内容。

### 可见区域截图
使用 Chrome 原生 API `captureVisibleTab()`，快速截图当前可见区域。

### 选定区域截图
提供交互式选择工具，用户可以用鼠标拖拽选择要截图的区域。

## 常见问题

**Q: 为什么某些网站的截图不完整？**
A: 某些网站可能有跨域限制。可以尝试使用"可见区域截图"功能。

**Q: 截图文件保存在哪里？**
A: 默认保存在浏览器的下载文件夹中。

**Q: 如何修改截图文件名？**
A: 可以在 `popup.js` 中修改 `downloadScreenshot` 函数中的文件名格式。
